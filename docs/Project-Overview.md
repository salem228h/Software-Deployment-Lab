# Project Overview

## Project Name

Enterprise Endpoint & Asset Management Lab

---

## Description

This project demonstrates enterprise endpoint administration using Windows 11 Enterprise and Windows Server 2022.

The lab simulates the daily responsibilities of an IT Support Technician responsible for workstation deployment, hardware inventory, software management, Windows security, PowerShell administration, and preventive maintenance.

---

## Objectives

- Deploy a Windows enterprise workstation
- Collect hardware inventory
- Document installed software
- Install standard business applications
- Verify Windows security
- Generate PowerShell reports
- Perform Windows maintenance
- Produce enterprise documentation

---

## Lab Environment

- Windows 11 Enterprise
- Windows Server 2022
- VirtualBox
- PowerShell
